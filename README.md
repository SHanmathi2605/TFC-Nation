# TFC Nation - Food Delivery Web Application

TFC Nation is a responsive food delivery web application built using React.js, TypeScript, Redux, and React Router.

## Features

- Dynamic food menu display
- Add and remove items from cart
- Checkout functionality
- Order analytics dashboard
- Pagination for menu items
- Responsive user interface
- State management using Redux
- API integration using Axios

## Technologies Used

- React.js
- TypeScript
- Redux
- React Router
- Axios
- Recharts
